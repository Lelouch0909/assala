# Test du Menu
## Fonctionnalités ajoutées :
1. **MenuButtons** - Menu avec 4 boutons qui apparaît à la fin de l'animation
   - Qui est Enis (en haut à gauche)
   - Ma lettre pour toi (en bas à gauche)
   - Enis à travers moi (en haut à droite)
   - Relancer l'animation (en bas à droite)
2. **ContentPage** - Pages de contenu (à compléter)
   - Template prêt pour ajouter du contenu
   - Bouton retour pour revenir à l'animation
   - Design cohérent avec le thème rose
3. **Navigation** - Gestion des transitions entre pages
   - Depuis la scène finale, le menu apparaît
   - Cliquer sur un bouton mène à la page correspondante
   - Le bouton "Relancer l'animation" redémarre depuis le début
   - Bouton retour pour revenir à la scène finale
## Comment ajouter du contenu :
Éditer le fichier `src/components/ContentPage.jsx` et modifier les sections selon le type de page.
